# Создаём реалистичных proxy-респондентов

Это приложение проводит автоматизированное интервью, используя 3 API-ключа от Groq для взаимодействия с моделями ИИ (в данном случае — `openai/gpt-oss-120b`). Оно читает вопросы из XML-файла ./interview/questions.xml, обрабатывает их через цепочку промптов и сохраняет логи в ./interview/interview_log.xml.

---

## Запуск автоматизированного пайплайна

Для запуска проекта убедитесь, что у вас установлен Python версии 3.10 или выше.

1. Создайте 3 API ключа от Groq: https://groq.com/. Расположите файлы с ними следующим образом:
```
./API keys/Construct API key.txt
./API keys/History Saver API key.txt
./API keys/Inland Empire API key.txt
```

2. Установите необходимые пакеты из `requirements.txt`:

```bash
pip install -r requirements.txt
```

3. Составьте список вопросов для глубинного интервью. Запишите их файл:
```
./interview/questions.xml
```
по заданному образцу.

4. Составьте базовый контекст, профиль респондента, и предметную область. Запишите их в файл:
```
./prompts/Prompt Skeleton.md
```
по заданному образцу. 

5. Запустите главный скрипт:

```bash
python main.py
```

6. Готово! Ожидайте ответов proxy-респондента в файле:
```
./interview/interview_log.xml
```
